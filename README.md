# Software Engineer Portfolio

A modern, responsive portfolio website built with Next.js 14, featuring a split-screen layout, interactive components, and smooth animations.

## Tech Stack

-   Next.js 14 (App Router)
-   TypeScript
-   Tailwind CSS
-   Framer Motion
-   Chart.js
-   Lucide Icons
